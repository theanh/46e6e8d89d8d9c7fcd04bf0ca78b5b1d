# app_survey
Application allows user to take a survey

# How to run
1. Install dependencies
2. Migrate database
3. Setup sample data
- Run these command:
  + bundle install --path vendor/bundle
  + bundle exec rake db:migrate
  + bundle exec rake db:setup
